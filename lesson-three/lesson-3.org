* OOP Is the Language of Professional Software
#+ATTR_REVEAL: :frag (appear)
  - Flutter's entire SDK is built with OOP — understanding classes
     is non-negotiable for reading Flutter docs
  - The factory constructor pattern (fromJson) is used in every app that talks to an API
  - Design patterns (MVC, MVVM, Clean Architecture) are all OOP patterns —
    used at Google, Meta, and Pathao
  - Dart's null-safe OOP is a safer version of Java/Kotlin OOP — concepts transfer directly to Android dev
  - AI code generators produce OOP-heavy code; understanding classes helps you read and modify
    AI output confidently
